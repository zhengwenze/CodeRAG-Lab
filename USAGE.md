# CodeRAG Lab 使用教程

## 系统简介

CodeRAG Lab 是一个基于 RAG（检索增强生成）技术的代码库智能问答系统。你可以用自然语言提问，系统会自动从代码库中检索相关代码片段，并生成带引用的回答。

---

## 快速开始

### 1. 访问系统

打开浏览器访问：**http://localhost:5173**

你会看到系统界面，主要包含三个功能模块：
- 💬 **问答** - 向 AI 助手提问
- 📚 **数据集** - 上传和管理文档
- 📊 **评测** - 测试系统性能

---

## 功能使用

### 💬 问答功能（核心功能）

这是系统的主要功能，让你可以通过自然语言了解代码库。

**使用步骤：**

1. 在底部的输入框中输入你的问题，例如：
   - "如何初始化一个 FastAPI 应用？"
   - "这个项目如何进行数据检索？"
   - "用户认证模块在哪里？"

2. 点击发送按钮或按 Enter 键

3. 系统会返回回答，其中：
   - **回答内容**：AI 根据检索到的代码片段生成的回答
   - **引用来源**：每段回答对应的文件路径和行号

**示例：**
```
问：如何创建一个新的数据集？

答：可以通过 POST /datasets 接口创建数据集...
引用来源：
- server/app/main.py:328 (行号)
- server/app/services/dataset.py:45
```

---

### 📚 数据集管理

用于上传文档（代码、文档、笔记等）到系统中，使系统能够基于这些内容回答问题。

**上传文档：**

1. 点击「数据集」菜单
2. 点击「上传文档」按钮
3. 选择文件（支持 PDF、Word、Markdown、TXT 等格式）
4. 等待系统解析和向量化

**查看文档：**

1. 在数据集列表中点击某个数据集
2. 可以看到该数据集下的所有文档
3. 点击文档可查看其分块（chunks）情况

---

### 📊 评测功能

用于评估系统的检索和问答效果。

**运行评测：**

1. 点击「评测」菜单
2. 点击「开始评测」按钮
3. 选择评测数据集（系统内置或自定义）
4. 等待评测完成

**查看结果：**

评测完成后会显示：
- **Hit Rate@K**：检索命中率
- **Citation Rate**：引用准确率
- **Recall**：召回率
- **MRR**：平均倒数排名

## API 调用示例

如果你是开发者，也可以直接调用 API：

### 健康检查
```bash
curl http://localhost:8000/health
```

### 问答
```bash
curl -X POST http://localhost:8000/chat \
  -H "Content-Type: application/json" \
  -d '{
    "messages": [{"role": "user", "content": "如何初始化 FastAPI？"}],
    "top_k": 5
  }'
```

### 上传数据集
```bash
curl -X POST http://localhost:8000/datasets \
  -F "file=@/path/to/file.pdf"
```

## 常见问题

### Q1: 回答显示"未找到相关内容"怎么办？
- 检查是否已上传相关文档到数据集
- 尝试用更精确的关键词提问

### Q2: 回答速度慢怎么办？
- 检查网络连接（使用云端 LLM 时）
- 考虑使用本地模型（llama.cpp）

### Q3: 如何更新代码库索引？
- 重新上传最新文档到数据集
- 系统会自动重新向量化

## 技术架构（可选了解）

- **前端**：Vue3 + Element Plus
- **后端**：FastAPI (Python)
- **向量库**：Qdrant
- **LLM**：MiniMax API (默认)
- **Embedding**：BAAI/bge-small-en-v1.5

### Markdown 渲染与代码高亮

- 前端输出的回答现在支持将 Markdown 直接渲染为 HTML，并对代码块进行语法高亮显示。
- 实现要点：使用 MarkdownIt 渲染 Markdown；使用 PrismJS 提供代码高亮；使用 DOMPurify 做输出过滤，防止 XSS。
- 验证方法：在后端返回包含 Markdown 的回答时，前端应正确显示格式化后的内容并高亮代码块。
- 快速测试示例：发送包含如下内容的回答文本：
  ```markdown
  ```js
  console.log('Hello CodeRAG')
  ```
  ```
- 如需进一步定制：可添加更多语言的 Prism 语言包、支持主题切换、或缓存渲染结果以提高性能。
